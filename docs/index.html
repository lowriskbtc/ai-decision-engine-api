Quick Start

Get started with the AI Decision Engine API in minutes:

  1. Get your API key from the dashboard
  2. Include it in the X-API-Key header
  3. Make your first request!

Base URL

Production: https://api.aiweedcompany.com
Development: http://localhost:8000

Example Request

import requests

headers = {
  "X-API-Key": "your_api_key_here",
  "Content-Type": "application/json"
}

response = requests.post(
  "http://localhost:8000/decisions/evaluate",
  json={"category": "FINANCIAL", "amount": 1000},
  headers=headers
)

Authentication

All API endpoints (except /health) require authentication using an API key.

API Key Header

Include your API key in the X-API-Key header:

X-API-Key: your_api_key_here

Getting an API Key

API keys are available in different tiers:

  • Free: 100 requests/month
  • Dev: 1,000 requests/month
  • Pro: 10,000 requests/month
  • Enterprise: 1,000,000 requests/month

Generate keys using the API Key Dashboard or contact support.

API Endpoints

GET /health

Check API health and status. No authentication required.

Response

{
  "status": "healthy",
  "version": "1.0.0",
  "timestamp": "2025-11-05T01:00:00",
  "autonomy_level": 77.4,
  "system": "operational"
}

POST /decisions/evaluate Free

Evaluate a decision using the AI decision-making framework.

Request Body

Parameter Type Required Description
category string Yes Decision category (STRATEGIC, OPERATIONAL, FINANCIAL, etc.)
amount number No Amount involved (default: 0)
description string No Description of the decision (max 1000 chars)

Example Request

{
  "category": "FINANCIAL",
  "amount": 1000.0,
  "description": "Invest in marketing campaign"
}

Response

{
  "id": "DEC_20251105010000",
  "risk_level": "LOW",
  "ai_can_decide": false,
  "action_required": "EXECUTE_IMMEDIATELY",
  "autonomy_level": 77.4
}

POST /risk/assess Pro

Assess risk level for a decision or action.

Request Body

Parameter Type Required Description
amount number No Amount involved (default: 0)
category string No Decision category (default: OPERATIONAL)
description string No Risk assessment description

Response

{
  "risk_level": "MEDIUM",
  "risk_score": 50,
  "recommendation": "Proceed with caution"
}

GET /autonomy/level

Get current AI autonomy level and system status.

Response

{
  "autonomy_level": 77.4,
  "target_autonomy": 100,
  "human_tasks": 4,
  "ai_tasks": 10
}

POST /autonomy/should-execute

Determine if AI should autonomously execute a task.

Request Body

Parameter Type Required Description
task_type string Yes Type of task (e.g., "operational", "financial")
risk_level string No Risk level (LOW, MEDIUM, HIGH, CRITICAL)

POST /memory/insights Pro

Get insights from the AI memory system.

Request Body

{
  "decision_data": {
    "category": "FINANCIAL",
    "amount": 1000
  }
}

Analytics Endpoints

Access usage statistics and analytics. Requires Pro or Enterprise tier.

GET /analytics/stats Pro

Get API usage statistics.

Query Parameters: days (default: 30)

GET /analytics/endpoint/{path} Pro

Get statistics for a specific endpoint.

Key Management

Manage your API keys programmatically. Requires Dev, Pro, or Enterprise tier.

GET /api/keys/list

List all API keys.

POST /api/keys/generate

Generate a new API key.

POST /api/keys/deactivate

Deactivate an API key.

Code Examples

Python

from api.client import AIDecisionAPIClient

client = AIDecisionAPIClient(
  api_key="your_key",
  base_url="http://localhost:8000"
)

result = client.evaluate_decision(
  category="FINANCIAL",
  amount=1000
)

JavaScript

const response = await fetch(
  'http://localhost:8000/decisions/evaluate',
  {
    method: 'POST',
    headers: {
      'X-API-Key': 'your_key',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      category: 'FINANCIAL',
      amount: 1000
    })
  }
);

cURL

curl -X POST http://localhost:8000/decisions/evaluate \
  -H "X-API-Key: your_key" \
  -H "Content-Type: application/json" \
  -d '{"category": "FINANCIAL", "amount": 1000}'
