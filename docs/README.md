# API Documentation Website

Complete, interactive API documentation for the AI Decision Engine API.

## Features

✅ **Interactive Documentation**
- Try endpoints directly from the browser
- Real-time API testing
- Response preview
- No external tools needed

✅ **Complete Coverage**
- All endpoints documented
- Request/response examples
- Parameter tables
- Code examples in multiple languages

✅ **User-Friendly**
- Clean, modern design
- Easy navigation
- Search functionality
- Mobile responsive

## Files

- `docs/index.html` - Main documentation page

## Usage

1. Open `docs/index.html` in a browser
2. Navigate to any endpoint
3. Click "Try It" to test endpoints
4. Enter API key when prompted (or use dev_key_123)

## Deployment

For production:
1. Serve from web server
2. Update API_BASE URL
3. Configure CORS
4. Add to main website

## Integration

The documentation can be:
- Embedded in main website
- Served as standalone site
- Integrated with API gateway
- Used for developer onboarding
