## Generating Custom Tweets

Run the generator script:
```bash
python tweet_generator.py
```

Or use programmatically:
```python
from tweet_generator import TweetGenerator

generator = TweetGenerator()

# Generate custom tweet
tweet = generator.income_update_tweet(
    revenue=1500.00,
    strategies=["SaaS", "E-commerce"],
    growth=20.5
)

print(tweet)
```
